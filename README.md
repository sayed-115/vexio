# Vexio Downloader

A blazing-fast, premium Web Application for natively downloading and archiving maximum-quality media from YouTube and other platforms. Powered by a lightweight FastAPI backend and `yt-dlp`.

## Features
* Stunning, dark-themed "Glassmorphism" Web UI
* Direct highest-quality 4K/8K fragment parsing
* Bypasses slow YouTube throttling with Aria native concurrent chunking 
* Native zero-loss Opus/M4A audio extraction
* Automated FFmpeg file merging directly into standalone .mp4

## Local Setup
Just run the `start.bat` file! It will dynamically build a secure virtual Python environment, install necessary dependencies without clashing, and boot up the local webserver at `http://127.0.0.1:5000`.
